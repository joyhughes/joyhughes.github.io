### Deploy Jen to Github Page
1. Run `npm run build` on `Jen/Lux/lux_react` to get the `dist` folder.
2. Copy the `dist` folder from Jen to the root path here
3. Run `chmod +x ./process_dist.sh` and then `./process_dist.sh`
4. `git add` and `git commit` the changes
5. Push this to remote to make a new deployment
